# yoda1-scale

A small utility to get data from the [Yoda1 Bluetooh Body Scale](https://www.aliexpress.com/item/1005002553230163.html).

## Requirements

- Go 1.15+
- Bluetooth scale linked above

## Build instructions

```sh
mkdir -p ./bin
go build -o ./bin/yoda1-scale ./cmd/yoda1-scale
./bin/yoda1-scale
```
